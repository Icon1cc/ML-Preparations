# 07 System Design for AI

This section is interview-focused and teaches how to answer AI/ML system design prompts with clear tradeoff reasoning.

## Interview approach
1. Clarify requirements.
2. Define success metrics and constraints.
3. Propose architecture by layers.
4. Explain tradeoffs.
5. Discuss failure modes and monitoring.
6. Present rollout strategy.
